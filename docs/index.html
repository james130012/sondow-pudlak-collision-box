Reproducible symbolic collision checkpoint

当前公开检查点为 fcce697-symbolic-collision-checkpoint： Lean（证明器）已在同一个 bigN（大阈值）上检查 upper.U bigN < measured bigN 与 measured bigN <= upper.U bigN 的形式化对撞。它不是具体数值 N（阈值）的求值结果，也不是无条件证明 γ 无理性的最终版本。

复刻命令： git checkout fcce697-symbolic-collision-checkpoint， 然后运行 lake env lean integration/SondowProjectMonth11Month12ProjectLengthTargetUpperEndpoint.lean。
